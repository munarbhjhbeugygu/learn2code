Python: Цикл while операторы

Тапсырма 1: 1-ден 5-ке дейінгі сандарды while циклын пайдаланып шығарыңыз.

# Сандарды шығарыңыз
i = 1
while i <= 5:
    print(i)
    i += 1

Тапсырма 2: 10-нан бастап 1-ге дейінгі сандарды while циклын пайдаланып кері ретпен шығарыңыз.

# Сандарды кері ретпен шығарыңыз
i = 10
while i > 0:
    print(i)
    i -= 1

Тапсырма 3: while циклын пайдаланып, 20-ға дейінгі барлық жұп сандарды шығарыңыз. 

# Жұп сандарды шығарыңыз
i = 2
while i <= 20:
    print(i)
    i += 2

Тапсырма 4: while циклын пайдаланып, сандарды енгізу арқылы олардың қосындысын табыңыз. Теріс сан енгізілгенде циклды тоқтатыңыз. 

# Қосындыны табыңыз
total = 0
while True:
    num = int(input())
    if num < 0:
        break
    total += num
print(total)

Тапсырма 5: Бір while циклында айнымалыны пайдаланып, 1-ден 50-ге дейінгі барлық тақ сандарды шығарыңыз. 

# Тақ сандарды шығарыңыз
i = 1
while i <= 50:
    if i % 2 != 0:
        print(i)
    i += 1
