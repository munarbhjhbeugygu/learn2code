C#: Шартты операторлар

Тапсырма 1: Берілген санның 5-тен үлкен немесе кіші екенін тексеріңіз. 

# Айнымалыны құрыңыз
number = 7

# 5-тен үлкен немесе кіші екенін тексеріңіз
if number > 5:
    print("Сан 5-тен үлкен")
else:
    print("Сан 5-тен кіші")

Тапсырма 2: Екі санды салыстырыңыз және қайсысы үлкен екенін анықтаңыз. 

# Екі айнымалыны құрыңыз
a = 10
b = 15

# Қайсысы үлкен екенін анықтаңыз
if a > b:
    print("a үлкен")
else:
    print("b үлкен")

Тапсырма 3: Үшінші санның ең үлкен екенін тексеріңіз. 

# Үш айнымалыны құрыңыз
x = 8
y = 14
z = 12

# Ең үлкен санды анықтаңыз
if x > y and x > z:
    print("x ең үлкен")
elif y > x and y > z:
    print("y ең үлкен")
else:
    print("z ең үлкен")

Тапсырма 4: Сіздің жасыңыз бойынша жеңілдік бар-жоғын анықтаңыз (жеңілдік 18 жасқа дейін және 65 жастан асқан адамдарға беріледі). 

# Айнымалыны құрыңыз
age = 20

# Жеңілдікті анықтаңыз
if age < 18 or age > 65:
    print("Жеңілдік бар")
else:
    print("Жеңілдік жоқ")

Тапсырма 5: Екі санның тең немесе тең емес екенін анықтаңыз. 

# Айнымалыларды құрыңыз
m = 9
n = 9

# Тең немесе тең емес екенін тексеріңіз
if m == n:
    print("Сандар тең")
else:
    print("Сандар тең емес")
