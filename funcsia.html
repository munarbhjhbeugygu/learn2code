# Функцияны жазыңыз
def greet():
    print("hello")

Тапсырма 2: Берілген санның квадратының мәнін қайтаратын функция жазыңыз.

# Квадрат мәнін қайтаратын функция жазыңыз
def square(x):
    return x ** 2

Тапсырма 3: Екі санның қосындысын қайтаратын функция жазыңыз. 

# Қосындыны қайтаратын функция жазыңыз
def add(a, b):
    return a + b

Тапсырма 4: Берілген жолды кері қайтарып беретін функция жазыңыз. 

# Жолды кері қайтаратын функция жазыңыз
def reverse_string(s):
    return s[::-1]

Тапсырма 5: Берілген тізімнің барлық элементтерін көбейтіп беретін функция жазыңыз. 

# Тізімнің элементтерін көбейтіп қайтаратын функция жазыңыз
def multiply_list(lst):
    result = 1
    for num in lst:
        result *= num
    return result
