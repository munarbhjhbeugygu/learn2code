Python: Цикл for операторы

Тапсырма 1: 1-ден 5-ке дейінгі сандарды басып шығару үшін for циклін қолданыңыз.

# Циклді қолданыңыз
for i in range(1, 6):
    print(i)

Тапсырма 2: Сандар тізімін (1, 2, 3, 4, 5) пайдаланып, олардың квадратының мәнін басып шығарыңыз.

# Сандар тізімін пайдалану
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num ** 2)

Тапсырма 3: 10-нан 1-ге дейінгі сандарды кері ретпен басып шығарыңыз.

# Кері ретпен басып шығару
for i in range(10, 0, -1):
    print(i)

Тапсырма 4: Сөзді (word) әр әріппен бірге жаңа жолға басып шығарыңыз. 

# Әр әріппен басып шығару
word = 'Python'
for char in word:
    print(char)

Тапсырма 5: 1-ден 10-ға дейінгі сандардың қосындысын есептеп, нәтижесін басып шығарыңыз. 

# Сандардың қосындысын есептеу
total = 0
for i in range(1, 11):
    total += i
print(total)
